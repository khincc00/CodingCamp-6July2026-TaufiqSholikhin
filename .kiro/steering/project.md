# Project: Pencatatan Pengeluaran

Ini adalah expense tracker web app sederhana untuk tugas CodingCamp RevoU.

## Stack
- HTML5, CSS3 (styles.css), Vanilla JavaScript (app.js)
- Tailwind CSS v4 via CDN
- localStorage untuk persistensi data
- Canvas API untuk donut chart

## File Utama
- `index.html` — versi final dengan Tailwind + glassmorphism
- `index_awal.html` — versi awal ditulis manual
- `styles.css` — semua styling dan CSS variables dark/light mode
- `app.js` — semua logika: CRUD, sort, custom category, tema

## Konvensi
- Bahasa Indonesia untuk semua label dan teks UI
- Tidak ada framework JS, tidak ada build tool, tidak ada backend
- Semua data disimpan di localStorage (client-side only)
